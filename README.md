 # Car Price Prediction with Flask

A machine learning-powered web application that predicts the resale price of used cars based on key features such as mileage, age, fuel type, and transmission. The application is built using Flask and deployed online for real-time predictions.

🔗 Live App: https://car-price-prediction-with-flask-1.onrender.com/

📂 GitHub Repository: https://github.com/Gladne/Car_Price_Prediction_with_Flask.git

# Project Overview

This project demonstrates an end-to-end machine learning workflow, including:

- Data preprocessing and feature engineering
- Model training using regression techniques
- Deployment of the trained model using Flask
- Interactive web interface for user input and predictions

The system allows users to input vehicle details and receive an estimated selling price instantly.

## Features
- Predicts used car prices in real time
- Simple and user-friendly web interface
- Handles categorical variables (fuel type, transmission)
- Model deployed using Flask and Gunicorn
- Lightweight and easy to run locally

## Machine Learning Workflow
- Data Preprocessing
- Model Integration
- Web Application (Flask)

## Tech Stack
- Frontend: HTML, CSS
- Backend: Flask (Python)
- Machine Learning: Scikit-learn
- Libraries: NumPy, Pandas
- Deployment: Gunicorn + Render

## Installation & Setup
1. Clone the Repository
```
git clone https://github.com/Gladne/Car_Price_Prediction_with_Flask.git
cd Car_Price_Prediction_with_Flask
```
2. Create Virtual Environment
```
python -m venv venv
source venv/bin/activate   # Linux/Mac
venv\Scripts\activate      # Windows
```
3. Install Dependencies
```
pip install -r requirements.txt
```
4. Run the Application
```
python mainapp.py
```
5. Open in Browser and paste the link
```
http://127.0.0.1:5001/
```
